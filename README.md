## 프로젝트 소개

* 프로젝트명: **Solive**
* 서비스 특징: 문제 별 1 대 1 화상 풀이 매칭 프로젝트
* 주요 기능
  - 회원 관리
  - 화상 문제 풀이 매칭
  - 게시판
* 주요 기술
  - WebRTC
  - WebSocket
  - JWT Authentication
  - REST API
* 참조 리소스
  * Tailwind CSS: 디자인 전반 적용
  * Material Tailwind: 디자인 전반 적용
  * OpenVidu: 오픈 소스 멀티 플랫폼 화상 회의 솔루션
  * JPA: 객체와 데이터베이스의 관계를 매핑
  * Querydsl: 쿼리 작성
* 배포 환경
